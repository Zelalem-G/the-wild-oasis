# 🏕️ The Wild Oasis

> A full-stack cabin rental management system built with **React**, **Supabase**, and **React Query**.  
> Designed for hotel staff to manage cabins, bookings, and guests efficiently — with secure authentication, real-time data, and a modern responsive UI.

## 🚀 Features

### 👥 Authentication & Roles
- 🔒 Secure login and signup restricted to hotel employees  
- 🧑‍💼 Role-based access (admin & staff)  
- 🖼️ Staff can upload avatars and update personal info  

### 🏡 Cabin Management
- 🧾 Table view of all cabins (photo, name, price, capacity, discount)  
- 🛠️ Add, edit, or delete cabins (with photo uploads via Supabase storage)  
- 📱 Responsive layout with modern React patterns  

### 📅 Booking Management
- 📖 Manage bookings with check-in, check-out, and deletion options  
- 💵 Handle payments (confirmation inside the app)  
- 🍳 Add breakfast or extra services at check-in  
- 🔍 Filter by booking status (unconfirmed / checked-in / checked-out)  
- ⚡ Real-time data updates synced with Supabase  

### 🧾 Guest Information
- 👤 Guest details: full name, email, nationality, ID, and flag  
- 🗒️ Observations & notes per guest  

### 📊 Dashboard & Analytics
- 📆 See guests checking in/out today  
- 📈 Charts for bookings, occupancy, and sales  
- 💹 Statistics for last 7, 30, or 90 days  
- ☕ Track total vs. breakfast-only sales  
- 📊 Stay duration metrics  

### ⚙️ App Settings
- 🥐 Adjustable breakfast price  
- 🕒 Minimum/maximum nights per booking  
- 👥 Maximum guests per booking  
- 🌙 Dark Mode toggle
